Mass Manufacturing Order Merger Odoo Apps

Merge Manufacturing Order Odoo App helps users to easily merge multiple MRP orders in one click. Users can select mass mrp orders and merge it with a single click, please note that selected orders must have the same state and same products and its bill of materials. Users can configure the default merge type for merge MRP orders that will apply in every merge order and also tick on "Notify in Chatter" option so it gives notification of merged orders into the chatter. Users can merge manufacturing orders with many operations and get Merged MRP orders notification into the chatter.

Features

Configuration for Default Merge Type

User can select the default type for merge manufacturing orders that will apply in every merge order.

Configuration for Notify in Chatter

If User can ticked the "Notify in Chatter" option then it gives notification of merged orders into the chatter.

New Order and Cancel or Delete Selected Orders

User can merge selected orders in new one then cancel or delete selected orders.

Merge Order on Existing Selected Order and Cancel or Delete Others

User have option to merge order with existing selected order and cancel or delete other orders.

Configuration for Merge MRP Order

Goto Manufacturing -> Configuration -> Settings then configure merge MRP order in that select default merge type and if you tick the "Notify in Chatter" option then it gives notification of merged orders into the chatter.

Merge Manufacturing Orders Menu

You can select manufacturing orders and click on "Merge Manufacturing Orders" menu.

New order and Delete all selected order

You can see merge manufacturing orders wizard with the default selected merge type.

Create a New Order and Delete Selected Orders

You can see a new order will be created and other selected orders are deleted.

You can see merged manufacturing order form view and in that merged orders notification comes into the chatter.

Merge Manufacturing Orders Menu

You can select manufacturing orders and click on "Merge Manufacturing Orders" menu.

Merge order on existing selected order and cancel others

You can see merge manufacturing orders wizard with selected merge type.

Merged order on existing selected order and cancelled others

You can see an another orders merged with selected existing order and rest of all selected orders are cancelled.

You can see merged manufacturing order form view and in that merged orders notification comes into the chatter.

Different State Order Error

Selected records state needs to be same either Draft, Confirmed or In Progress otherwise it will raise an error.

Different Product/Bill of Material Orders Error

You can only merge same product/bill of material manufacturing orders otherwise it will raise an error.
